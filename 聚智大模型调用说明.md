# 聚智大模型调用说明

## **概述**

聚智大模型是一个**兼容 OpenAI API 协议的内部网关**，通过 `openai` SDK 发起请求，但需要走**自定义的 HMAC-SHA256 认证签名**来生成 `api_key`。

> [!IMPORTANT] 模型选择不依赖 OpenAI SDK 的 `model` 参数，而是通过认证签名中的 `modelId` 来指定。SDK 中的 `model` 参数仅作占位符使用。
> 

---

## **调用流程**

```
准备认证参数 → 构造签名字符串 → HMAC-SHA256 签名 → 组装 auth_string → Base64 编码生成 api_key → 创建 OpenAI 客户端 → 调用 chat.completions.create
```

### **1. 签名生成**

使用 HMAC-SHA256 对以下拼接字符串签名，密钥为 `api_secret`：

```
host: {host}
date: {date}
{request_line}
```

| 字段 | 来源 | 示例 |
| --- | --- | --- |
| `host` | 从 `request_url` 解析 | `xxx.xxx.com` |
| `date` | GMT 格式当前时间 | `Thu, 20 Mar 2026 01:44:13 GMT` |
| `request_line` | 固定格式 | `POST /openapi/flames/api/v1/openai/chat HTTP/1.1` |

### **2. 构建认证字符串**

将签名与其他参数组装成结构化字符串，然后 **Base64 编码**作为最终的 `api_key`：

```
hmac api_key="{api_key}", algorithm="hmac-sha256",
headers="host date request-line", signature="{signature}",
modelId="{model_id}", modelSource="{model_source}",
traceId="{trace_id}", host="{host}",
date="{date_str}", request-line="{request_line}"
```

### **3. 发起请求**

```python
client = OpenAI(api_key=auth_base64, base_url=request_url)

response = client.chat.completions.create(
    model="xxx",           # 占位符，不影响模型选择
    messages=[
        {"role": "user", "content": content}
    ],
    stream=False           # 支持 stream=True 流式调用
)
```

## **所需参数**

### **认证参数（必填）**

| 参数 | 说明 | 环境变量 |
| --- | --- | --- |
| `request_url` | 聚智网关 API 地址 | `OPENAI_API_BASE` |
| `api_key` | API 密钥 ID | `OPENAI_API_ID` |
| `api_secret` | API 密钥 Secret（用于 HMAC 签名） | `OPENAI_API_SECRET` |
| `model_id` | 模型唯一标识（**真正决定调用哪个模型**） | `MODEL_ID` |
| `model_source` | 模型来源标识 | `MODELSOURCE` |
| `trace_id` | 请求追踪 ID | `TRACE_ID` |

### **可选配置参数**

| 参数 | 说明 | 环境变量 | 默认值 |
| --- | --- | --- | --- |
| 请求超时 | 单次请求超时时间（秒） | `LLM_REQUEST_TIMEOUT` | `120` |
| 总超时 | 总超时时间（秒） | `LLM_TOTAL_TIMEOUT` | `300` |
| 最大重试 | 最大重试次数 | `LLM_MAX_RETRIES` | `3` |
| 最大 Token | 最大生成 Token 数 | `LLM_MAX_TOKENS` | 无限制 |

## **请求内容格式**

### **纯文本**

```python
content = "你好"
```

### **多模态（图文混合）**

```python
content = [
    {
        "type": "image_url",
        "image_url": {"url": "图片地址"}
    },
    {
        "type": "text",
        "text": "图中描绘的是什么景象?"
    }
]
```

---

## **模型选择机制**

与标准 OpenAI API 不同，聚智网关的模型选择**不依赖** SDK 的 `model` 参数：

| 参数 | 作用 | 是否影响模型选择 |
| --- | --- | --- |
| `model_id`（在认证签名中） | 指定模型唯一 ID | ✅ **是** |
| `model_source`（在认证签名中） | 指定模型来源 | ✅ **是** |
| `model`（SDK 参数） | OpenAI SDK 必填占位符 | ❌ **否** |
